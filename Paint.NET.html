Paint.NET

Paint.NET

Операційна система: Windows 7, Windows Vista з SP1, Windows XP з SP3

Розробники: dotPDN, LLC.

Мова програмування: C#, C++

Перший випуск: 2004 р.

Paint.NET – піксельний графічний перетворювач з великим набором інструментів для редагування фотографій для Windows (також підходить для Windows xp, Windows 7 і Windows 10), базується на .NET Framework.
Програма створена спеціально для Microsoft Windows під управлінням Microsoft в Університеті Вашингтоні. Утиліта має багато функцій платних редакторів, і в цьому є перевага перед аналогами. Додаток безкоштовний для роботи та придбання.
Даний графічний редактор є оптимальним для сучасних технологій, це покращує продуктивність. Інтерфейс має схожість із Photoshop. Додаток: вміє взаємодіяти з кількома файлами одночасно; працює з шарами, тобто можна розробити картинку шляхом об'єднання різних частин або зображень; зручне у використанні; веде історію дій користувача з перетворення зображень у спеціальному вікні.
При інтенсивному використанні програми необхідно стежити за наявністю вільного місця на карті пам'яті. Програма задовольняє вимоги щодо основної редакції фотографій, наприклад, тон, різкість, колір корекція і більш поглибленим, таким як виробництво нових картинок.
